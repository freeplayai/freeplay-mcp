# Changelog

Notable additions, fixes, or breaking changes to the Freeplay MCP server.

## [0.0.3]

Initial experimental release.

This MCP server is experimental and will change. Current limitations:

- Does not support deployment operations or destructive deletion actions
- Uses your regular Freeplay API key (not specially scoped for agents)

See [README](https://github.com/freeplayai/freeplay-mcp#readme) for installation and usage.
